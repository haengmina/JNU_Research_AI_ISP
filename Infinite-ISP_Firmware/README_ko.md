# Infinite-ISP
Infinite-ISP는 알고리즘 개발부터 RTL 설계, FPGA/ASIC 구현, 그리고 관련 펌웨어 및 도구에 이르기까지 ISP 개발 전 과정을 아우르는 풀스택 ISP 개발 플랫폼입니다. 이 통합 플랫폼은 ISP 개발자들이 보다 빠르게 혁신을 이룰 수 있도록 지원합니다. Python으로 작성된 완전한 카메라 파이프라인 모듈 컬렉션, Kria KV260 개발 보드에서 파이프라인을 구현하기 위한 FPGA 비트스트림 및 관련 펌웨어, 그리고 마지막으로 파이프라인용 독립형 Python 기반 튜닝 도구 애플리케이션을 포함하고 있습니다. Infinite-ISP 프로젝트의 주요 구성 요소는 다음과 같습니다:

| 저장소 이름        | 설명      | 
| -------------  | ------------- |
| **[Infinite-ISP_AlgorithmDesign](https://github.com/xx-isp/infinite-isp)**                        | 알고리즘 개발을 위한 Infinite-ISP 파이프라인의 Python 기반 모델 |
| **[Infinite-ISP_ReferenceModel](https://github.com/10xEngineersTech/Infinite-ISP_ReferenceModel)**                         | 하드웨어 구현을 위한 Infinite-ISP 파이프라인의 Python 기반 고정 소수점(fixed-point) 모델 |
| **[Infinite-ISP_FPGABitstream](https://github.com/10xEngineersTech/Infinite-ISP_FPGABitstream)**                                      | Kria KV260의 Xilinx® ZU5EV Ultrascale FPGA를 위한 FPGA 비트스트림 |
| **[Infinite-ISP_Firmware](https://github.com/10xEngineersTech/Infinite-ISP_Firmware)**  :anchor:                                    | Kria KV260의 내장 Arm® Cortex®A53 프로세서를 위한 펌웨어 |
| **[Infinite-ISP_Tuning Tool](https://github.com/10xEngineersTech/Infinite-ISP_TuningTool)**                              | Infinite-ISP를 위한 캘리브레이션 및 분석 도구 모음 |
